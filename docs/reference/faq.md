## 1. 일반적인 질문

### Q: Bespoke AI Suite란 무엇인가요?
**A:** Bespoke AI Suite는 SMB(중소기업)와 스타트업을 위한 AI 기반 콘텐츠 생성 및 마케팅 자동화 플랫폼입니다. 텍스트, 이미지, 비디오 등 다양한 형태의 콘텐츠를 AI로 생성하고, 마케팅 캠페인을 자동화할 수 있습니다.

### Q: 어떤 규모의 기업에 적합한가요?
**A:** 주로 다음과 같은 기업에 적합합니다:
- 직원 수 10-500명의 SMB(중소기업)
- 콘텐츠 마케팅에 집중하는 스타트업
- 마케팅 자동화가 필요한 e-commerce 업체
- 제한된 리소스로 마케팅 효율성을 높이고 싶은 기업

### Q: 다른 AI 콘텐츠 도구와 무엇이 다른가요?
**A:** Bespoke AI Suite의 차별점:
- **통합 플랫폼**: 콘텐츠 생성부터 캠페인 관리까지 원스톱 솔루션
- **멀티 에이전트 시스템**: Crew AI 기반의 협업 AI 에이전트
- **SMB 맞춤 설계**: 중소기업의 요구사항에 특화된 UI/UX
- **유연한 API**: 기존 워크플로우와 쉬운 통합
- **비용 효율성**: 엔터프라이즈급 기능을 합리적인 가격에 제공

### Q: 무료 체험이 가능한가요?
**A:** 네, 14일 무료 체험을 제공합니다. 신용카드 등록 없이 주요 기능을 체험해볼 수 있으며, 체험 기간 중 최대 50개의 콘텐츠를 생성할 수 있습니다.

## 2. 계정 및 인증

### Q: 계정을 어떻게 생성하나요?
**A:** 계정 생성 방법:
1. [thebespoke-ai.com](https://thebespoke-ai.com)에서 "무료 체험 시작" 클릭
2. 이메일, 회사명, 업종 정보 입력
3. 이메일 인증 완료
4. 초기 설정 마법사 진행

### Q: 소셜 로그인을 지원하나요?
**A:** 현재 다음 소셜 로그인을 지원합니다:
- Google 계정
- Microsoft 계정
- GitHub 계정 (개발자용)

### Q: 2단계 인증(2FA)을 설정할 수 있나요?
**A:** 네, 보안 강화를 위해 2FA를 적극 권장합니다:
- Google Authenticator
- Authy
- Microsoft Authenticator
- SMS 인증 (백업용)

### Q: 비밀번호를 잊어버렸어요.
**A:** 비밀번호 재설정 방법:
1. 로그인 페이지에서 "비밀번호 찾기" 클릭
2. 등록된 이메일 주소 입력
3. 이메일로 받은 재설정 링크 클릭
4. 새 비밀번호 설정

## 3. 콘텐츠 생성

### Q: 어떤 종류의 콘텐츠를 생성할 수 있나요?
**A:** 현재 지원하는 콘텐츠 유형:
- **텍스트**: 블로그 포스트, 소셜 미디어 포스트, 이메일, 광고 문구
- **이미지**: 제품 이미지, 소셜 미디어 그래픽, 광고 배너
- **비디오**: 짧은 프로모션 비디오, 소셜 미디어 비디오 (베타)

### Q: 한국어 콘텐츠 생성이 가능한가요?
**A:** 네, 다음 언어를 지원합니다:
- 한국어 (완전 지원)
- 영어 (완전 지원)
- 일본어, 중국어 (간체/번체), 스페인어, 프랑스어 (제한적 지원)

### Q: 브랜드 가이드라인을 적용할 수 있나요?
**A:** 네, 브랜드 일관성을 위해 다음을 설정할 수 있습니다:
- 브랜드 톤앤매너
- 컬러 팔레트
- 로고 및 브랜드 에셋
- 글쓰기 스타일 가이드
- 금지 단어/표현 목록

### Q: 생성된 콘텐츠의 품질은 어떻게 보장되나요?
**A:** 품질 보장 시스템:
- **AI 품질 점수**: 각 콘텐츠에 0-100점 품질 점수 제공
- **다중 에이전트 검토**: Research, Planning, Creation, Review 에이전트가 협업
- **A/B 테스트**: 여러 버전을 생성하여 최적 선택
- **인간 피드백**: 사용자 평가를 통한 지속적 학습

### Q: 콘텐츠 생성에 얼마나 시간이 걸리나요?
**A:** 콘텐츠별 생성 시간:
- **짧은 텍스트** (소셜 미디어): 10-30초
- **긴 텍스트** (블로그 포스트): 1-3분
- **이미지**: 30초-2분
- **비디오**: 3-10분 (길이에 따라)

## 4. 캠페인 관리

### Q: 마케팅 캠페인을 어떻게 생성하나요?
**A:** 캠페인 생성 단계:
1. **목표 설정**: 캠페인 목적과 KPI 정의
2. **타겟 설정**: 고객 페르소나 및 타겟 채널 선택
3. **콘텐츠 계획**: AI가 콘텐츠 캘린더 생성
4. **승인 및 실행**: 콘텐츠 검토 후 스케줄링
5. **모니터링**: 실시간 성과 추적

### Q: 어떤 마케팅 채널을 지원하나요?
**A:** 현재 지원 채널:
- **소셜 미디어**: Facebook, Instagram, Twitter, LinkedIn, YouTube
- **이메일 마케팅**: Mailchimp, SendGrid 연동
- **광고 플랫폼**: Google Ads, Facebook Ads (베타)
- **웹사이트**: WordPress, Shopify 플러그인

### Q: 캠페인 자동화가 가능한가요?
**A:** 네, 다음과 같은 자동화를 지원합니다:
- **스케줄링**: 최적 시간대 자동 포스팅
- **A/B 테스트**: 자동 성과 비교 및 최적화
- **대응 자동화**: 댓글/메시지 자동 응답
- **성과 기반 조정**: 실시간 데이터에 따른 전략 수정

### Q: 캠페인 성과를 어떻게 측정하나요?
**A:** 제공하는 성과 지표:
- **참여도**: 좋아요, 댓글, 공유, 클릭률
- **도달률**: 노출 수, 도달 수, 브랜드 인지도
- **전환율**: 리드 생성, 매출 전환, ROI
- **품질 점수**: 콘텐츠 품질, 브랜드 일관성

## 7. 기술적 문제

### Q: 브라우저 호환성은 어떻게 되나요?
**A:** 지원 브라우저:
- **완전 지원**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **제한적 지원**: Internet Explorer 11 (기본 기능만)
- **모바일**: iOS Safari 14+, Android Chrome 90+

### Q: 업로드 파일 크기 제한이 있나요?
**A:** 파일 크기 제한:
- **이미지**: 최대 10MB, JPG/PNG/GIF
- **동영상**: 최대 100MB, MP4/MOV/AVI
- **문서**: 최대 5MB, PDF/DOC/TXT
- **배치 업로드**: 동시에 최대 50개 파일

### Q: 데이터를 내보낼 수 있나요?
**A:** 데이터 내보내기 옵션:
- **콘텐츠**: JSON, CSV, ZIP 형태로 일괄 다운로드
- **캠페인 데이터**: 성과 보고서 Excel/CSV 내보내기
- **API 로그**: 개발자를 위한 요청/응답 기록
- **계정 데이터**: GDPR 준수 전체 데이터 패키지

### Q: 속도가 느려요. 어떻게 해결하나요?
**A:** 성능 개선 방법:
1. **브라우저 캐시 삭제**: Ctrl+Shift+Delete (Chrome)
2. **다른 브라우저 시도**: 최신 Chrome/Firefox 권장
3. **인터넷 연결 확인**: 최소 10Mbps 권장
4. **애드블로커 비활성화**: 일부 기능 차단 가능
5. **지원팀 문의**: 여전히 느리다면 기술 지원 요청

## 8. 데이터 및 개인정보

### Q: 내 데이터는 안전한가요?
**A:** 보안 조치:
- **암호화**: 전송/저장 시 AES-256 암호화
- **접근 제어**: 역할 기반 권한 관리
- **감사 로그**: 모든 접근/변경 기록
- **정기 백업**: 실시간 백업 및 재해 복구
- **보안 인증**: SOC 2 Type II, GDPR 준수

### Q: AI 학습에 내 데이터가 사용되나요?
**A:** 데이터 사용 정책:
- **명시적 동의**: 학습 데이터 사용은 선택사항
- **익명화**: 개인 식별 정보 제거 후 사용
- **제외 옵션**: 언제든지 학습에서 제외 가능
- **투명성**: 사용된 데이터 유형 명시

### Q: GDPR 준수는 어떻게 되나요?
**A:** GDPR 준수 사항:
- **데이터 주체 권리**: 접근, 수정, 삭제, 이동권 보장
- **처리 근거**: 명확한 법적 근거 명시
- **데이터 보호 담당자**: 전담 DPO 배치
- **침해 신고**: 72시간 내 침해 신고 체계
- **프라이버시 바이 디자인**: 설계 단계부터 개인정보 보호

### Q: 데이터를 완전히 삭제할 수 있나요?
**A:** 삭제 옵션:
- **개별 삭제**: 콘텐츠, 캠페인 개별 삭제
- **계정 삭제**: 30일 후 완전 삭제
- **백업 삭제**: 90일 후 백업에서도 완전 제거
- **삭제 증명서**: 요청 시 삭제 완료 증명서 발급
